Nostr Key Converter

⚠️ SECURITY WARNING ⚠️

  • Never enter your private keys (nsec) on any website, including this one!
  • This tool should only be used offline on a secure computer
  • Anyone with access to your private key can steal your funds
  • Download this tool and run it locally for private key operations

About This Tool

  • This tool converts Nostr keys to Bitcoin addresses and formats
  • All conversion is done locally in your browser - no data is sent to servers
  • For public keys (npub):
    • Legacy addresses start with "1" and work with all Bitcoin wallets
    • Native SegWit addresses start with "bc1" and offer lower fees
  • For private keys (nsec):
    • WIF format is supported by most Bitcoin wallets
    • Always handle private keys with extreme caution
